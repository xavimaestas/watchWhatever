Instructions

Movies: search movie in Google search, go to main IMDB page.
Grab "tt" ID from URL, ex: tt1300854, input into Movies bar
and press enter for movie link. Use an ad blocker and click until
it plays. If you get a 404, they don't have the movie yet. If it's
lagging, try a different source from the drop down.

TV Shows: search TV shows in Google Search, go to main IMDB page.
Grab "tt" ID from URL, ex: tt1300854, input into Movies bar.
Input Season and Episode you want in appropriate box. For single
values do not put a "0" before the numbers. Press enter after entering
all the values to generate your link. See tips on watching above.
